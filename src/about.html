About this server

A tiny test site to exercise request parsing, header creation and binary file streaming in your custom HTTP server.

What to check

  1. Correct MIME type (Content-Type)
  2. Accurate Content-Length header
  3. Proper CRLF line endings in headers

Advanced

Try curl with verbose output: curl -v http://localhost/gallery.html

Tip: put these files under ./src (or whatever your server root is) so requests like /index.html map correctly.
